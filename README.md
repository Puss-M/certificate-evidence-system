# 暑期实训：可信证书存证系统

本仓库用于完成暑期实训创新项目 1：基于区块链的实训证书与学业证明存证系统。当前定位为：

> 证书生命周期管理系统 + 哈希存证验真系统。

主线闭环包括模板配置、Excel 导入、证书 PDF 与二维码生成、SHA-256 哈希、本地哈希链或测试链存证、扫码验真、撤销补发和审计日志。DID/VC、Merkle Root、多机构协同和可信简历属于后续扩展，不阻塞当前交付。

## 当前工程

```text
backend/                 FastAPI 后端、数据库模型、证书与验真服务、自动化测试
frontend/                Vue 3 管理员端
docs/                    原始材料、提交模板、填写稿、项目规划和协作记录
scripts/                 项目辅助脚本
outputs/、logs/          本地生成物和运行日志，不提交 Git
```

后端已覆盖管理员端主要接口、证书生成、本地哈希链回执和公共验真；前端已完成管理员端基础页面。学生端身份隔离、证书列表和详情接口仍是待完成项，不能按已上线功能演示。

## 当前优先级

1. 先完成发证、下载、扫码、验真和撤销闭环。
2. 优先使用 SHA-256 + 本地哈希链保证稳定演示。
3. 确保证书编号、PDF、二维码、哈希、回执和状态可互相追踪。
4. 提高项必须通过独立适配层接入，不扩大主线范围。
